# ZN Sleepwear - Locally runnable demo

Open `index.html` in a browser to view the soft-pink product showcase. The demo uses a `mailto:` flow to create an email with the order details (no backend).

Quick steps:

1. Dubbelklik op `index.html` of open het bestand in je browser.
2. Voeg items toe aan de winkelwagen.
3. Klik `Bestelling plaatsen`, vul je gegevens en klik `Bevestig en e-mail` om je mailclient te openen met de bestelling.

If you want a proper online ordering flow, I can add a simple server endpoint (Node/Express) or integrate with a service like Formspree.
